# Dataset

This is a starting dataset for the forces project containing vulnerabilities written in C and, their equivalent written in Rust.

> both version were written by hand without relying on transpilers

## Structure

To compile all programs : `make` at the root of the project

# Disclaimer on the use of LLMs

The use of LLMs was kept to the minimum

- Querying the documentation
- Compilation errors
- ...

to avoid adding bias to the dataset
